DAOlytics

participation analytics tools for DAOs

Through this tool, people can use freely available POAP distribution data to create interaction networks that describe the participation and interaction of members of a DAO. The tool then uses these networks to study the network architecture so that the DAO can gain insight in how it is functioning and what could be done to further improve it. - - allow user to define threshold value for edges which is then used to update the network features - add scientific overview of the relevance of the provided network features and how they can be used to improve the DAO

input

All matrices for individual activities can be summed over a predefined time period. This results in a network that, between every possible pair of active DAO members, indicates in how many overlapping activities they participated.
